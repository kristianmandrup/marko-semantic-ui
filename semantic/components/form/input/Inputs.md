Inputs
------

### Checkbox

Create a pre-checked checkbox for `male` gender.

`ui-checkbox(name='male' checked='true' label='Male' tab='1')`

### Dropdown

Create a dropdown for `skills`.

`drop-down(name='skills' tab='1' list='data.user.skills')`
